Install Instructions:

Prerequisites: Both the CLI and generated project have dependencies that require Node 6.9.0 or higher, together with NPM 3 or higher.

1. Install Node 6.9.0 or higher
2. Install npm 3 or higher
3. open cmd
4. type on cmd "npm install -g @angular/cli"
5. Clone solution on file system
6. navigate to project folder (.i.e MusicStore) on cmd, using "cd MusicStore" command.
7. npm install
8. type  on cmd "ng serve"
9. open chrome browser and type url: http://localhost:4200

To run unit tests:
1. navigate to project folder (.i.e MusicStore) on cmd, using "cd MusicStore" command.
2. type  on cmd "ng test"
